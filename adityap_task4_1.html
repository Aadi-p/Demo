# Basic Structure of C++

The basic structure of a C++ program typically follows a specific format, consisting of various key components.


1. #Include iostream:

It is a Header file libary that lets us work with input and output object,Such as a Cout used in the line.


2. Std::cout:

Std::cout is used to output (or print) data to the standard output,which is typically the terminal or console. It is part of the C++ Standard Library. How ever if we use (using namespace std;) in the second line we can just use cout instead of: std::

using namespace std; means that we can use names for object and variables from the standard libary.


3. Int main:

This is Called a function. (Any codes inside its Curly Bracket will be Executed).


4. Return 0(zero):

Ends the main function. when ever the return type of a function is not void,we return some value. In this example: Return Type is int(integer) and we return 0.


*T/N: Every c++ Statement ends with an Semi colon: [Using namespace std;].

Do not forget to add the Closing Curly bracket to Actually end the main Function.
